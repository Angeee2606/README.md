# ClassBuddy App


## 📱 About the App
ClassBuddy is a school organizer app designed to help students stay organized, productive, and connected. Features include homework tracking, test scheduling, group chat, and a unique “Teacher Sync.”


## 🎯 Key Features
- 📅 Task and Schedule Management  
- 📝 Homework Reminders  
- 📊 Test and Exam Schedule  
- 💬 Group Project Chat  
- 🔄 Teacher Sync Feature  
- 🌙 Dark Mode Interface
